# PRD: AV 资源搜索下载器 (AV Downloader)

## 1. 项目背景
为用户提供一个一站式的 AV 资源搜索、元数据预览及下载管理工具，解决搜索源分散、资源质量参差不齐、下载步骤繁琐的问题。

## 2. 核心功能点 (Refined)

### 2.1 聚合搜索 (Multi-source Search)
- **多源抓取**: 同时支持 JAVBus、JAVDB 等主流站点。
- **元数据展示**: 包含封面图、标题、发行日期、演员、番号（ID）。
- **磁力解析**: 自动抓取页面内的磁力链接，解析文件大小、磁力名称。

### 2.2 智能筛选与排序 (Smart Filtering)
- **优先级策略**: 
  - 1. 带中文字幕优先（关键词如 `-C`, `字幕`, `CN`）。
  - 2. 高清优先（根据文件大小降序）。
- **过滤机制**: 自动去重，过滤失效或无磁力的条目。

### 2.3 下载集成 (Download Integration)
- **远程管理**: 集成 Aria2 RPC 或 Transmission RPC，支持一键推送到 NAS/远程下载器。
- **本地唤起**: 提供磁力链接直达，可一键唤起本地 BitComet、迅雷等。
- **任务监控**: 在 Web UI 中查看下载进度（仅限 Aria2/Transmission）。

### 2.4 用户界面 (UI/UX)
- **现代化设计**: 基于 Bootstrap 5，采用响应式卡片布局 (Responsive Cards)。
- **交互体验**: 
  - 搜索联想/历史记录。
  - 图片懒加载。
  - 暗色模式支持。
- **性能**: 限制最大宽度 1400px，保证大屏展示效果。

### 2.5 基础架构与配置
- **代理支持**: 可选配置本地代理 (如 127.0.0.1:7890) 以访问被屏蔽站点。
- **跨平台**: 基于 Python Flask，易于部署在 Windows/macOS/Linux。

## 3. 验收标准 (Acceptance Criteria)
1. 用户输入番号后，能在 5 秒内展示来自至少两个来源的搜索结果。
2. 搜索结果按“中文字幕 > 文件大小”逻辑自动排序。
3. 点击“下载”按钮能成功发送任务到 Aria2 或弹出磁力链接唤起提示。
4. UI 界面在 1920x1080 和移动端均能正常显示。

## 4. 技术栈
- **后端**: Python 3.12 + Flask + BeautifulSoup4 (或 Playwright)
- **前端**: HTML5 + Bootstrap 5 + FontAwesome
- **下载**: 磁力链接协议协议唤起 (Magnet Protocol)
 / Transmission RPC
